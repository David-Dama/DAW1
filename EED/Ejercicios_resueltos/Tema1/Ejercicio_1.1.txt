¿Qué es un programa informático?

    Según la Rae es un conjunto de instrucciones que permite a un ordenador u otro aparato realizar una 
    determinada función de forma automática.
    En otras palabras, una serie de pasos a seguir que perimte que un dispositivo electronico haga algo,
    ya sea una orden o cualquier otra cosa. A esa serie de pasos se le conoce como algortimo.


Menciona dos funciones o tareas que puede realizar un programa.

    Un programa puede automatizar tareas para poder reducir el tiempo que invrtimos en ellas y también puede
    gestionar y organizar información a gusto del solicitante.


¿Es lo mismo un programa que el software? ¿O el software es algo más amplio? 

    Un programa y el software no es lo mimo ya que el software se refiere a algo más amplio debido a que puede 
    recoger más de un programa informático. Para comprenderlo mejor, un programa es simplemente una secuencia 
    de instrucciones, y seguir instrucciones es una de las partes del software, pero tiene otras más.

    Sistemas: Conjunto de elementos interrelacionados entre sí para lograr un mismo objetivo
