6. Escribe una definición de una línea para cada categoría de código
    
    Código fuente: Conjunto de instrucciones escritas por el programador en un lenguaje de programación legible (como Python o Java).

    Código objeto: Versión intermedia del programa, traducida por el compilador a un lenguaje que entiende la máquina pero aún no ejecutable.

    Código ejecutable: Programa final completamente traducido y listo para ser ejecutado directamente por el sistema operativo.
